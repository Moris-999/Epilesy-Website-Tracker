# Epilesy-Website-Tracker
🧠 Epilepsy Tracker Web App

A web app to track seizures, manage medications, and get basic AI-based seizure risk predictions.

Features
Authentication – Login/Register using local storage
Dashboard – Overview cards for seizures & medications
Medication Tracker – Add and view medications
Seizure Logger – Record date, duration, triggers, notes
AI Predictor – Predicts HIGH or LOW seizure risk
Analytics – Charts showing seizure trends and triggers
Tech Stack
HTML, CSS, JavaScript
Chart.js for data visualization
LocalStorage for data persistence
